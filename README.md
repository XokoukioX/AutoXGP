## AutoXGP
！！！严禁滥用！！！后续微软如果修改策略我也不保证会继续维护这个项目！！

使用方法：

1. [前往微软开发中心下载驱动](https://developer.microsoft.com/en-us/microsoft-edge/tools/webdriver/)
2. 将下载的'msedgedriver.exe'移动到本目录
3. 打开终端运行以下代码
```
# 下载库
pip install -r requirements.txt
# 运行
python3 autoXGP.py
```
